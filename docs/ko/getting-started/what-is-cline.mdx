---
title: "Caret이란 무엇인가요?"
description: "VS Code에서 AI 기반 개발을 돕는 Caret에 대한 소개입니다."
---

Caret은 Microsoft Visual Studio Code와 통합되는 AI 개발 어시스턴트입니다. IDE와 LLM(대규모 언어 모델) 사이의 인터페이스를 제공하여 코드 개발을 촉진하고 생산성을 높이며 새로운 개발자의 진입 장벽을 낮춥니다. 권한에 따라 Caret은 파일을 읽고 쓰거나, 명령을 실행하고, 웹 브라우저를 사용하며, 모델 컨텍스트 프로토콜(MCP) 서버를 통해 기능을 확장할 수 있습니다.

Caret이 특별한 이유는 코드 생성에 대한 신중한 접근 방식과 광범위한 통합 기능 때문입니다. 단순히 코드 스니펫을 생성하는 것을 넘어, Caret은 솔루션을 단계별로 계획하고 전체 개발 환경을 인지하며 모든 변경 사항에 대한 명시적인 승인을 요구함으로써 개발자와 협력합니다. Caret은 대규모 코드베이스를 이해하고, 새로운 엔지니어의 온보딩을 가속화하며, 모델 컨텍스트 프로토콜 마켓플레이스를 통해 수백 가지 도구와 연결되어 간소화된 프로젝트 배포부터 자동화된 인시던트 대응까지 모든 것을 자연어 명령으로 가능하게 합니다.
